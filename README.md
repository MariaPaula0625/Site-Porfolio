# Portfolio Premium

Este projeto é um portfólio pessoal moderno desenvolvido para apresentar experiências, habilidades, projetos e informações profissionais de forma visual e responsiva.

O site foi construído utilizando React com Vite, focando em desempenho, organização de componentes e animações modernas.

## Tecnologias Utilizadas

### Front-end

* React
* Vite
* JavaScript (ESModules)
* JSX
* HTML5
* CSS3

### Estilização

* Tailwind CSS
* PostCSS
* Autoprefixer

### Bibliotecas

* Framer Motion — animações modernas e interativas
* Lucide React — ícones SVG modernos

### Ferramentas de Desenvolvimento

* ESLint
* React Hooks
* React Refresh

# Como Executar o Projeto

## Clone o repositório

```bash
git clone https://github.com/MariaPaula0625/Site-Porfolio.git
```

## Acesse a pasta

```bash
cd Site-Porfolio
```

## Instale as dependências

```bash
npm install
```

## Execute o projeto

```bash
npm run dev
```

O projeto ficará disponível em:

```bash
http://localhost:5173
```

---

# Build de Produção

```bash
npm run build
```

Para visualizar a build:

```bash
npm run preview
```
